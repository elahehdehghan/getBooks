پروژه گرفتن کتاب از OpeenLibrary با استفاده از api

ویژگی ها:

1- جستجوی کتاب‌های موضوع مورد نظر

2- فیلتر کتاب‌های منتشر شده بعد از سال ۲۰۰۰

 3- ذخیره خودکار در فایل CSV

4- صفحه بندی خودکار با ذخیره شماره صفحه

5- مدیریت خطا و اتمام صفحات

روش استفاده:

بار اول: ۵۰ کتاب ذخیره می‌شه.

دفعات بعد: صفحه بعدی پردازش می‌شه.

وقتی صفحات تموم بشه، از صفحه ۱ شروع می‌کنه.

خروجی فایل books.csv شامل نام کتاب ، نویسنده و سال انتشار است.

خروجی فایل page.txt شماره ی صفحه ی بعدی که خوانده می شود، است.
